# Email verifing algorithm written in C.
How to compile.

`
$ git clone https://github.com/byteofsoren/email_validator.git
`

`
$ cd email_validator
`

`
$ mkdir build
`

`
$ cd build
`

`
$ cmake ..
`

`
$ make
`

## How to use.

`
$ emailValidator test@test.com
`

If the return is 1 its a valid email.
